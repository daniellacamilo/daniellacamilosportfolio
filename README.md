The **Personal Portfolio Project** is a **modern professional website**, developed with **HTML, CSS, and JavaScript**, with the goal of presenting information about the developer, their projects, and contact details in a clear, interactive, and responsive manner.

The project consumes dynamic data from the **GitHub API**, allowing information such as profile and repositories to be loaded automatically, keeping the content always up-to-date.

------

## Features

- Page structure developed with **semantic HTML**

- Modern styling with **CSS**, using:

- CSS variables

- Animations

- Responsive layout (desktop, tablet, and mobile)

- Integration with the **GitHub API** for:

- Dynamic display of profile information

- Automatic listing of repositories

- Display of projects in an **interactive carousel** using **Swiper.js**

- **Contact form with frontend validation**, ensuring correct field completion

- Dedicated **form submission confirmation** page

- Fluid navigation with fixed menu and smooth scrolling

- Intuitive and organized interface, focused on user experience

## Running Locally

To run the project locally, follow the steps below.

### Prerequisites

- Visual Studio Code (https://code.visualstudio.com/) (or another editor of your choice)

- **Live Server** extension installed in VS Code

### Steps

1. Clone the repository:

``bash

git clone https://github.com/your-username/your-repository.git

```

2. Access the project folder:

``bash

cd your-repository

```

3. Open the project in Visual Studio Code:

``bash

code .

```

4. Open the `index.html` file, right-click and select **"Open with Live Server"**.

The website will open in the browser and all changes can be viewed in real time.
